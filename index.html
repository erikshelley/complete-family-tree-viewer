Complete Family Tree Viewer

A browser-based application for visualizing genealogy data from GEDCOM files. Load a local GEDCOM file, select a root individual, and generate a complete family tree including all relatives and in-laws for that individual.

For a full description of the features and how to use this application, please see the GitHub Repository. You can also report any issues or request new features there.
